# Plan 04-05: Ad Monitoring Automation

## Wave Overview
- **Phase:** 04 - TikTok & Viral Analysis for Social Scout
- **Wave:** 5 of 5
- **Focus:** Create automated competitor ad monitoring cron workflow
- **Estimated Time:** 45 minutes

## Goal
Automatically monitor competitors for ad activity changes and send push notifications when they start running ads.

---

## Tasks

### Task 5.1: Create Ad Monitoring Cron Workflow
**Source Todo:** `2026-01-28-create-ad-monitoring-cron-workflow.md`
**Priority:** High

Create new workflow: **Nemo - Competitor Ad Monitor**

### Workflow Structure

```
[Schedule Trigger] → [Get Monitored Competitors] → [Loop] →
  [Execute Social Scout] → [Compare Ad Status] →
  [If Changed to TRUE] → [Send Push Notification] → [Update Status]
```

---

### Node Configurations

#### Node 1: Schedule Trigger
```json
{
  "parameters": {
    "rule": {
      "interval": [{ "field": "minutes", "minutesInterval": 30 }]
    }
  },
  "type": "n8n-nodes-base.scheduleTrigger",
  "typeVersion": 1.2,
  "position": [-400, 0],
  "id": "GENERATE_NEW_UUID",
  "name": "Every 30 Minutes"
}
```

---

#### Node 2: Get Monitored Competitors
```json
{
  "parameters": {
    "operation": "getAll",
    "tableId": "competitors",
    "returnAll": true,
    "matchType": "allFilters",
    "filters": {
      "conditions": [{
        "keyName": "monitor_ads",
        "condition": "eq",
        "keyValue": "true"
      }]
    }
  },
  "type": "n8n-nodes-base.supabase",
  "typeVersion": 1,
  "position": [-200, 0],
  "id": "GENERATE_NEW_UUID",
  "name": "Get Monitored Competitors",
  "credentials": {
    "supabaseApi": { 
      "id": "JbItbwVcQiGCLFAC", 
      "name": "NemoAIDatabase" 
    }
  }
}
```

#### Node 3: Loop Through Competitors
```json
{

  "typeVersion": 3,
  "position": [0, 0],
```

#### Node 4: Execute Social Scout Sub-workflow
```json
{
  "parameters": {
    "source": "database",
    "workflowId": "HCV-51qLaCdcxHGx2yBcO",
    "mode": "each",
    "options": {},
    "workflowInputs": {
      "values": [
        { "name": "target ", "value": "={{ $json.url }}" },

  "type": "n8n-nodes-base.executeWorkflow",
  "typeVersion": 1.2,
  "position": [200, 0],

  "name": "Run Social Scout"

```

---

#### Node 5: Compare Ad Status (If Node)
```json
{
  "parameters": {
    "conditions": {
      "options": {
        "caseSensitive": true,
        "leftValue": "",
        "typeValidation": "strict"
      },
      "conditions": [
        {
          "leftValue": "={{ $json.json_output.ads_running }}",
          "rightValue": true,
          "operator": { 
            "type": "boolean", 
            "operation": "equals" 
          }
        },
        {
          "leftValue": "={{ $('Loop Competitors').item.json.last_ads_status }}",
          "rightValue": false,
          "operator": { 
            "type": "boolean", 
            "operation": "equals" 
          }
        }
      ],
      "combinator": "and"
    }
  },
  "type": "n8n-nodes-base.if",
  "typeVersion": 2.3,
  "position": [400, 0],
  "id": "GENERATE_NEW_UUID",
  "name": "Ads Just Started?"
}
```

**Logic:** Notify only when:
- Current `ads_running = true`
- Previous `last_ads_status = false` (or null)

## Test Scenarios

| Scenario | Expected Behavior |
|----------|-------------------|
| No competitors with monitor_ads = true | Workflow completes with no actions |
| Competitor has monitor_ads = true, ads_running = false | No notification, status updated |
| Competitor ads change false → true | Push notification sent |
| Competitor ads stay true | No notification (no change) |
